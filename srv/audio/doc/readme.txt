Способ вызова описаны в каталоге doc/test/

Настройка сервиса:
===== eq_ausrv04.json =====
{
"BIND": "0.0.0.0",
"PORT": "32105",
"USERNAME": "username",
"PASSWORD": "password",
"SETMODE": "1"
}
==========================
где, BIND, PORT, USERNAME, PASSWORD - параметры web-сервиса;
SETMODE - 0 - проговорить все, как в URL, исключая неизвестные слова;
          1 - проговорить после доп.обработки URL - отделяем число 
 от строки из букв. Например, "v_kabinet,21A" преобразуем
 в "v_kabinet,21,A"
